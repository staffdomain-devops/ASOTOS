| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | scripts/lib/api_client.py | mitigate | API-specific _is_retryable_* functions only retry 429 and 5xx; 4xx permanent failures are not retried, preventing runaway retry loops on bad credentials |
| T-02-02 | Information Disclosure | scripts/lib/dlq_writer.py | mitigate | error_message truncated to [:2000] before writing; prevents full stack traces with secrets from appearing in DLQ artifact |
| T-02-03 | Denial of Service | scripts/lib/api_client.py | mitigate | stop_after_delay(60) cap on all retry decorators prevents runaway retries from exceeding the GitHub Actions job timeout |
| T-02-04 | Tampering | prompt_template.md | accept | File is static text in the repo; no user input reaches it at write time. Jinja2 StrictUndefined is enforced at render time in Phase 3. |
</threat_model>
